## 微信听写APP
听写单词、句子的微信WEB应用。

## 界面
+ 主页，展示总录音数信息。
+ 录音，在移动微信中点击，开始录音，制作单词/句子的录音。
+ 列表，展示录音列表，编辑录音对应的文字内容。
+ 测试，随机抽 {n} 个录音，组成试卷，在固定时间内完成测试。

### 技术选型

**前端**
+ dva
+ antd-mobile

**后端**
+ SSH
+ MySQL
+ Apache
